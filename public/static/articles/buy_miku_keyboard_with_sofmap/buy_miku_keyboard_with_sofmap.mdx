昨年12月にTwitterで発表され、
待ちに待った[REALFORCE ✕ 初音ミク GX1 Keyboard](https://www.realforce.co.jp/news/20240119/20240119_REALFORCE_Miku_GX1_News_Release.pdf)でしたが、
最安値の[Amazonから売り出されてるやつ](https://amzn.to/3Ueu51t)(42900円)は、

そこで、

(僕の勝手な皮算用なので、信じるか信じないかはあなた次第です)

## お得に買う方法

お得に買う方法はズバリ、以下の方法を全部使うことです！

羅列すると、

- 39歳以下限定ですが...[JCBW](https://www.jcb.co.jp/jcb_nyukai/syokaicp_os202312.jsp?brand=1323054+3950+100.html&route=2399&takeone=9999999&campaign=J00000035&webone=kl2J8oU84rSpj3XZVNLdBXr0pXVaNGvccn%2BGm0GNNpY%3D)でAmazonギフト券を買う
- [楽天 Rebates](https://www.rebates.jp/sofmap?referrerid=NPtaJzH5o%2Bc%3D) を経由する
- [ソフマップ](https://click.linksynergy.com/deeplink?id=/CQoDa*vFDg&mid=37641&murl=https%3A%2F%2Fwww.sofmap.com%2Fproduct_detail.aspx%3Fsku%3D100739277)で買う
- [Amazon Pay](https://pay.amazon.co.jp/using-amazon-pay/evergreen?scmedium=paymentspage&place=banner)で[Amazonギフト券](https://amzn.to/3OI8xXN)支払いする

の4つです。

### JCBWを使う

[JCBW](https://www.jcb.co.jp/jcb_nyukai/syokaicp_os202312.jsp?brand=1323054+3950+100.html&route=2399&takeone=9999999&campaign=J00000035&webone=kl2J8oU84rSpj3XZVNLdBXr0pXVaNGvccn%2BGm0GNNpY%3D)
は、39歳以下限定のJCBが発行しているクレジットカードです。
これの初回キャンペーンを用いると、
Amazon.co.jpで使用した分の20%(最大10000円)が還元されます。

また、[この](https://www.jcb.co.jp/jcb_nyukai/syokaicp_os202312.jsp?brand=1323054+3950+100.html&route=2399&takeone=9999999&campaign=J00000035&webone=kl2J8oU84rSpj3XZVNLdBXr0pXVaNGvccn%2BGm0GNNpY%3D)
リンクを踏んで申し込むことで、1500円分の還元があります。(ただし即時発行に対応していないので、予約締め切り直前の場合は即時発行をお勧めします)

そして、
JCBWは[ポイントアップ登録](https://original.jcb.co.jp/shop/000052)をすることができ、
Amazon.co.jpでのお支払いが2.0%ポイント還元されます。

JCBWを上記リンクから契約し、
ポイントアップ登録をした上で、
Amazon.co.jpで[Amazonギフト券](https://amzn.to/3OI8xXN)を購入することで、

```text
47660 * 0.2 + 47660 * 0.02 + 1500
= 9532 + 953 + 1500
= 11985
```

11985円還元されます。

### 楽天 Rebatesを使う

[楽天 Rebates](https://www.rebates.jp/sofmap?referrerid=NPtaJzH5o%2Bc%3D)を経由してソフマップでお買い物をすると、1.0%ポイント還元されます (2024/01/28現在)。

また、[この](https://www.rebates.jp/sofmap?referrerid=NPtaJzH5o%2Bc%3D)リンクから登録すると、
初回購入で500円分のポイントが還元 (3000円以上の購入が必要) されます。

そのため、
楽天 Rebatesを上記リンクから登録し、
ソフマップで購入することで、

```text
47660 * 0.01 + 500
= 476 + 500
= 976
```

976円還元されます。

### ソフマップでAmazon Payで支払う

[ソフマップ](https://click.linksynergy.com/deeplink?id=/CQoDa*vFDg&mid=37641&murl=https%3A%2F%2Fwww.sofmap.com%2Fproduct_detail.aspx%3Fsku%3D100739277)
で購入すると、購入価格の10%がポイント還元されます。

この時点で、Amazonよりも安く買えるのですが、
さらにお得にするために、
[Amazon Pay](https://pay.amazon.co.jp/using-amazon-pay/evergreen?scmedium=paymentspage&place=banner)で支払うことで、
追加でAmazonギフト券を1.0%還元することができます。

アカウント登録をした上、
ソフマップでAmazon Payのギフト券支払いをすることで、

```text
47660 * 0.1 + 47660 * 0.01
= 4766 + 476
= 5242
```

5242円還元されます。

## まとめ

1. [JCBW](https://www.jcb.co.jp/jcb_nyukai/syokaicp_os202312.jsp?brand=1323054+3950+100.html&route=2399&takeone=9999999&campaign=J00000035&webone=kl2J8oU84rSpj3XZVNLdBXr0pXVaNGvccn%2BGm0GNNpY%3D)に申し込む
2. JCBWの[ポイントアップ登録](https://original.jcb.co.jp/shop/000052)をする
3. JCBWで[Amazonギフト券](https://amzn.to/3OI8xXN)を購入する
4. [楽天 Rebates](https://www.rebates.jp/sofmap?referrerid=NPtaJzH5o%2Bc%3D)を経由する
5. ソフマップで買う
    1. [日本語キーボード](https://click.linksynergy.com/deeplink?id=/CQoDa*vFDg&mid=37641&murl=https%3A%2F%2Fwww.sofmap.com%2Fproduct_detail.aspx%3Fsku%3D100739277)はこちら
    2. [英語キーボード](https://click.linksynergy.com/deeplink?id=/CQoDa*vFDg&mid=37641&murl=https%3A%2F%2Fwww.sofmap.com%2Fproduct_detail.aspx%3Fsku%3D100751227)はこちら
6. Amazon Payのギフト券支払いをする

という手順で、18203円安く購入する方法を紹介しました。

ソフマップは47660円と他と比べて高いですが、
これらの技を使うことで、
最安値だったAmazonの42900円よりもはるかに安い29457円で購入することができます。

僕は今からこれで買ってきます。
