(メモ)

Parallels Desktop の MacOS VM を実行してる時に実行元の PC 本体をスリープすると時刻がズレてくので、ズレないようにする。

cron で root 権限を用いて以下のコマンドを実行すれば良い。(時刻合わせのコマンド)

`sntp -sS time.asia.apple.com`

コマンドに落とし込むとこう。

```bash
sudo crontab -e

###### 以下を出てきた Vim に追記する ######
* * * * * sntp -sS time.asia.apple.com
```
